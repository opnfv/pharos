*************************
ZTE SH POD3 Specification
*************************


Introduction
------------

POD3(means ZTE-POD3) uses Fuel as the installer and performs os-nosdn-kvm-ha CI latest verification.
Feature projects like NFV-KVMV, OVSNFV will be run in this POD.
